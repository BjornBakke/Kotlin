# Integrasjon-person

Spring Boot-applikasjon for personsøk.

## Forutsetninger

- Java 25
- Maven

## Starte applikasjonen

```bash
mvn spring-boot:run
```

## Teste API-et

Når applikasjonen kjører, åpne nettleseren eller bruk curl:

```bash
curl http://localhost:8080/api/
```

Dette returnerer en liste med personer i JSON-format.
